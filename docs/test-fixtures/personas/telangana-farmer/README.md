- **EPS-95** — farmers are not EPF-registered.

**Bottom line:** highest single-scheme value (Rythu Bima ₹5 L), and shows IGNWPS
appearing when there's no state widow-pension to absorb it (contrast with Bihar
and Karnataka, where it's suppressed).
